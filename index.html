Help

This area will be filled with context-sensitive help content as you use different tools.

Below select how confident you are in your selections. Select missing if you believe that the point you are being asked to mark is biologically missing, for example, when a fin isn't there. Select unsure if the point is probably there, but you are uncertain of its exact location, for example, due to lighting or quality issues with the photo. Otherwise select OK.
